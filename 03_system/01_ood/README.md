# 面向对象设计OOD

> 考核关键是如何设计class，class的properties和methods定义好，掌握class的继承


## 1. 面试过程
- clarification，先理清`业务流程`
- 根据流程图，找到`core object`，明确`use case`
- 完善整体设计`class diagram`，写出class和重点method
