# 全栈面试指南

> 🎯 这是一本面向海外、外企、大厂CS相关岗位的面试指南，旨在帮助你系统的准备技术面试。

## 📚 关于本书

本书内容涵盖机器学习、数据科学和软件工程等多个领域的面试准备。无论你是准备面试 Software Engineer、Machine Learning Engineer、Data Scientist 还是 AI Engineer，都能在这里找到相应的准备指南。

### 🎯 适用人群

- Software Engineer (SDE)
- Machine Learning Engineer (MLE)
- Data Scientist (DS)
- Applied Scientist (AS)
- AI Engineer
- Data Engineer

### 💡 面试要点

- **逻辑通顺**: 清晰表达思路，展示解决问题的能力
- **框架思维**: 系统性地分析和解决问题
- **突出重点**: 突出个人优势和项目亮点
