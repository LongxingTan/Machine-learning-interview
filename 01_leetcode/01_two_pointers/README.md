# 双指针


- 同向双指针
  - 移除元素类
  - 滑动窗口
    - 数组类，用于正数，如果有负数采用hash/前缀和或DP.
      - 模式一：记录问题. 一般两个while, 或: 右指针for 左指针while收缩
      - 模式二：优化问题
    - 字符串类，用hash和计数器记录窗口内状态
  - 快慢指针类
- 相向双指针
  - 模式与二分查找类似，每次明确可以只移动一个
