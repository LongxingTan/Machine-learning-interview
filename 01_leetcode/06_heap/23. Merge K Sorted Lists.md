# 23. Merge K Sorted Lists
[https://leetcode.com/problems/merge-k-sorted-lists/](https://leetcode.com/problems/merge-k-sorted-lists/)


## solution

```python
import heapq

class Solution:
    def mergeKLists(self, lists: List[Optional[ListNode]]) -> Optional[ListNode]:
        res = []
        for head in lists:
            while head:  # 空间没有得到优化, 也没有利用排序的特点
                heapq.heappush(res, head.val)
                head = head.next
        
        dummy = head = ListNode(-1)
        while res:
            head.next = ListNode(heapq.heappop(res))
            head = head.next

        return dummy.next
```
时间复杂度：O(nlog(k)) <br>
空间复杂度：O(k)


```python
class Solution:
    def mergeKLists(self, lists: List[Optional[ListNode]]) -> Optional[ListNode]:
        # ListNode.__lt__ = lambda self, other: self.val < other.val  # 最简洁的做法是增加属性, 可以把node直接push进heap
        
        heap = []
        for i, l in enumerate(lists):
            # 注意只有首个进入head
            if l:
                heapq.heappush(heap, (l.val, i))  # python3 heap中前一个相等就会判断下一个, 因此想要把node进入heao, 需要3个元素
        
        
        dummy = head = ListNode()
        while heap:
            val, i = heapq.heappop(heap)
            node = lists[i]
            head.next = node
            head = head.next

            if node.next:             
                heapq.heappush(heap, (node.next.val, i))
                lists[i] = node.next

        return dummy.next
```
时间复杂度：O(n*k*log k) <br>
空间复杂度：O(k)
